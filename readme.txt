A clean and lightweight plugin to manage and display short-form reflective posts (pensées) with AJAX navigation and simple styling.

== Description ==

**Pensées** is a minimal plugin that registers a custom post type (`pensee`) and provides a shortcode viewer with AJAX-powered next, previous, and random navigation.

It includes:

- A `pensee` post type
- A shortcode `[pensee_viewer]` to display one pensée with controls
- AJAX endpoints for seamless post navigation
- Simple and responsive default styling

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`.
2. Activate the plugin through the Plugins menu in WordPress.
3. Use the `[pensee_viewer]` shortcode in a page or post.

== Frequently Asked Questions ==

= Can I customize the styles? =

Yes — the plugin includes a minimal CSS file which you can override in your theme.

= Is this translation-ready? =

Yes, it includes a text domain.
